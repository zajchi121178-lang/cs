Выполнен анализ данных о COVID-19 с использованием Apache Spark (PySpark) для решения 3 задач:

3\. \*\*Task 3: Дельта новых случаев в России за последнюю неделю марта 2021\*\*  

&#x20;  В выходном датасете: `число`, `кол\_во\_вчера`, `кол\_во\_сегодня`, `дельта`.

\## ⚙️ Технические особенности реализации



\- \*\*Среда:\*\* Windows 10/11, Python 3.13, Apache Spark 3.x.

\- \*\*Проблема среды:\*\* На Windows нативные библиотеки Hadoop могут вызывать ошибку `UnsatisfiedLinkError`.

\- \*\*Решение:\*\* Для гарантированной стабильности запись финальных результатов реализована через стандартный модуль Python `csv` (вместо `df.write.csv`). Вся логика агрегации, группировки и оконных функций выполнена средствами Spark SQL.

\- \*\*Особенности Git:\*\* Папка `results` добавлена в `.gitignore` для исключения тяжёлых данных из истории коммитов. Финальные CSV-файлы добавлены принудительно (`git add -f`) специально для сдачи работы, чтобы эксперт мог сразу проверить результаты без повторного запуска Spark-скрипта.
